# 🚀 Logic Box

A Python project that combines **Pattern Generation** and **Number Analysis** using fundamental programming concepts such as loops, conditional statements, and match-case.

---

## 📖 Overview

This project demonstrates the use of:

✅ While Loop  
✅ For Loop  
✅ Match-Case  
✅ If-Else Statements  
✅ Nested Loops  
✅ Range Function  
✅ Pattern Generation  
✅ Number Analysis  
✅ Break, Continue, Pass

---

## ✨ Features

### 🔺 Pattern Generator
- Accepts the number of rows from the user.
- Generates a right-angled triangle pattern.
- Uses nested loops for pattern creation.


### 🔢 Number Analyzer
- Accepts start and end values from the user.
- Determines whether each number is Odd or Even.
- Calculates the total sum of numbers in the range.

### 🚪 Exit Option
- Safely terminates the program.


### 🎯 Menu Driven Program

1. Pattern Generator
2. Number Analyzer
3. Exit
