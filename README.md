## Logic Box

## Overview
This project demonstrates:
- while loop
- for loop
- match-case
- if-else
- Pattern Generation
- Number Analysis

## Features

### 1. Generate a Pattern
- User enters number of rows.
- Program prints a triangle pattern using nested loops.

### 2. Analyze a Range of Numbers
- User enters start and end values.
- Program identifies odd/even numbers.
- Calculates the sum of all numbers.

### 3. Exit
- Ends the program safely.
